This repository includes MediaWiki extensions
that are not included in [the collection of all MediaWiki extensions](https://gerrit.wikimedia.org/g/mediawiki/extensions)
or in [The Super-repository of non-Wikimedia hosted extensions](https://github.com/MWStake/nonwmf-extensions).

The list of extensions is taken from [Category:Extensions not in ExtensionJson](https://www.mediawiki.org/wiki/Category:Extensions_not_in_ExtensionJson).

This repository is created for reference.
